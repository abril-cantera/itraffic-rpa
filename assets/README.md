# Asset Files for Email Classifier Add-in

This directory contains the icons and images used by the Outlook Add-in.

## Required Icons

- `icon-16.png` - 16x16px - Ribbon button (small)
- `icon-32.png` - 32x32px - Ribbon button (medium)
- `icon-64.png` - 64x64px - Ribbon button (large)
- `icon-128.png` - 128x128px - App listing, store

## Specifications

- Format: PNG with transparency
- Style: Simple, recognizable at small sizes
- Colors: Use #0078D4 (Microsoft blue) as primary color
- Subject: Email classification icon (e.g., envelope with AI sparkle)

## Temporary Placeholders

For development, we're using simple colored squares. Replace these with proper icons before production deployment.

## Design Tools

Recommended tools for creating icons:
- Figma (free, web-based)
- Adobe Illustrator
- Canva (templates available)

## Icon Guidelines

Follow Microsoft's design guidelines:
https://learn.microsoft.com/en-us/office/dev/add-ins/design/add-in-icons
